# preventDefault

#### You can pass `preventDefault` and prevent the default behaviour of the browser. This will prevent the default only if the state has changed, this way we are not over taking other event listeners

In this example we are hooking to the `contextmenu` event (usually the right mouse button) but we will prevent it from opening the context menu (only if state has changed).
<Playground >
  <Trap event="contextmenu" preventDefault>
    {(trapped, ref) => <Box innerRef={ref} isFocused={trapped} />}
  </Trap>
</Playground>
